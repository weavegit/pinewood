1) Create a DB called Pinewood

2) Open Visual Studio 2022 and click Tools -> Nuget Package Manager -> PCM

3) In the PCM, ensure the Default Project is set to PinewoodAPI project and type in add-migration (give it a name), then update-database.

4) Set Startup project as PinewoodUI.

5) Right-click on the PinewoodApi project and select debug ->  Start new instance

6) Now Start the PinewoodUI project.
